# LearningPlaywrightJS

PlaywrightJS examples and exercises used for learning Playwright in JavaScript.

Structure:
- `chapter_01_Basics/` — basic examples and setup verification
- `chapter_02_JS_Concepts/` — JavaScript concepts (placeholder)

Getting started:

1. Install dependencies:

```bash
npm install
```

2. Run Playwright tests (example):

```bash
npx playwright test
```
